# Portfolio Website Features

## Overview
This document describes the main features of Hamza Al-Ahdal's futuristic portfolio website.

---

## 1. Hero Section
- Animated name reveal with typing effect
- Dynamic tagline showcasing role as "AI & Full-Stack Developer"
- Call-to-action buttons (View Projects, Contact Me)
- Floating particle background animation

## 2. About Section
- Personal summary with glassmorphism card design
- Animated skill tags that float on hover
- Education details with timeline animation
- Profile image with neon glow border

## 3. Projects Showcase
- Interactive project cards with 3D tilt effect on hover
- Each card displays:
  - Project title and description
  - Technologies used as glowing badges
  - Links to demo/source code
- Projects included:
  - RAG AI Chatbot (Python, Gemini, FAISS)
  - AI Financial Data Chatbot (SerpApi, Llama 3)
  - Bank Management System (Node.js, MongoDB)

## 4. Skills Section
- Animated skill bars with percentage indicators
- Categorized skills:
  - Programming: Java, JavaScript, SQL, Python
  - Databases: MySQL, MongoDB
  - Frameworks: Node.js, Express.js
  - DevOps: Git, Docker, Kubernetes
- Hover effects with neon glow animations

## 5. Contact Section
- Contact form with glassmorphism styling
- Social links (LinkedIn, Email, Phone)
- Animated submit button with ripple effect

## 6. AI Chatbot Agent (Gemini 2.0 Flash)
- Floating chat bubble in bottom-right corner
- Expandable chat window with futuristic design
- **Powered by Google Gemini 2.0 Flash API** for intelligent responses
- **Animated Robot Avatar** (80px, centered):
  - Blinking cyan eyes
  - Mouth animates when "talking" (responding)
  - Eyes pulse when "thinking" (processing)
  - Magenta glow when "listening" (voice input)
  - Subtle wave animation on hover
- **Voice Features**:
  - Speech recognition (click mic to speak)
  - Text-to-speech (bot speaks responses aloud)
  - Works in Chrome, Edge, and modern browsers
- **AI Knowledge Base:**
  - Complete profile, education, and skills
  - All projects with live demo links
  - Availability and contact information
  - Maintains conversation context
- Typing indicator animation
- Message history with smooth scroll
- Fallback responses if API unavailable
