# AI Chatbot Feature

## Description
An intelligent AI-powered chat agent **powered by Google Gemini 2.0 Flash** that represents Hamza and can interact with potential clients and recruiters.

## Capabilities

### 1. Gemini AI Integration
- Connected to Google Gemini 2.0 Flash API
- Comprehensive system prompt with Hamza's complete profile
- Maintains conversation context across messages
- Fallback to local responses if API unavailable

### 2. Identity & Background
- Introduces itself as Hamza's AI assistant
- Provides information about education at Altinbas University
- Shares personal summary and career goals

### 3. Skills Discussion
- Lists programming languages and proficiency levels
- Explains framework and tool experience
- Discusses AI/ML expertise (Gemini, Llama 3, RAG, FAISS)

### 4. Project Showcase
- Describes each project with live demo links
- RAG AI Chatbot, Alhadaf Level, La Liga Hub
- Explains technologies used and achievements

### 5. Contact Assistance
- Provides contact information (email, phone, LinkedIn)
- Emphasizes availability for opportunities
- Shares location information

## User Experience
- Appears as floating bubble in corner
- Expands to full chat interface on click
- **Animated Robot Avatar** (80px, centered in header):
  - Eyes blink periodically for lifelike effect
  - Mouth animates (opens/closes) when responding
  - Eyes pulse when "thinking" during processing
  - Magenta glow when "listening" to voice input
